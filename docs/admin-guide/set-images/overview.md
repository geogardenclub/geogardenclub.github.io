# Set Images

<img width="300" src="/img/admin-guide/set-images.png"/>

The Set Images page allows you to update the images for a chapter, crop, and variety.

The `Set Images for Chapter` will search through all the gardens in the chapter and pick a random garden image to use as the chapter image. 

The `Set Images for Crop` will search through all the observations for the crop in the system and pick a random image to use as the crop image. 

The `Set Images for Variety` will search through all the observations for the variety in the system and pick a random image to use as the variety image.
