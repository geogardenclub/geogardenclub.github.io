# Refresh Caches

<img width="300" src="/img/admin-guide/refresh-caches.png"/>

There are four types of caches in GGC:
1. Chapters have cached garden names, crop names, usernames, badge names, and variety names. Pressing the `Refresh Chapter Caches` button will refresh the chapter caches for all chapters.
2. Gardens have cached crop IDs, variety IDs, years, number of plantings, and a crop quantity map. Pressing the `Refresh Garden Caches` button will refresh the garden caches for all gardens.
3. Gardeners have cached crop IDs, and variety IDs. Pressing the `Refresh Gardener Caches` button will refresh the gardener caches for all gardeners.
4. Geolocations for postal codes in the chapter. There are two ways to refresh geolocation caches:
   - Pressing the `Refresh All Geo Caches` button will refresh the geolocation caches for all postal codes in the chapters. This is a time-consuming operation.
   - Selecting a chapter from the chapter drop down then pressing the `Refresh Chapter Geo Cache` button will refresh the geolocation caches for the given chapter. Again this is a time-consuming operation.
