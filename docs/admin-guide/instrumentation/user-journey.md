# User Journey

<img width="300" src="/img/admin-guide/instrumentation/user-journey.png"/>

User Journey shows the sequence of events (in reverse chronological order) for a single user. 

Note that you can use [Set Retrieval Dates](set-retrieval-dates.md) to limit the range of events shown.
