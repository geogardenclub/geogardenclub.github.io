# Set Retrieval Dates

<img width="300" src="/img/admin-guide/instrumentation/set-retrieval-dates.png"/>

By default, all events are retrieved from the database. This can impact performance. You can use this command to specify the range of dates to be retrieved from the database. 

Then you can further limit the events displayed using the Date Picker in the [Event Table](event-table.md) command.

This command also effects the [Event Log](event-log.md) and [User Journey](user-journey.md) commands.
