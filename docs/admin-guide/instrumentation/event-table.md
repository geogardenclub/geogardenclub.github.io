# Event Table

<img width="300" src="/img/admin-guide/instrumentation/event-table.png"/>

The Event Table allows us to see the events for a given date range, organized by user. 

This enables us to see what individual users are doing over a given time span.

Note that you can use the [Set Retrieval Dates](set-retrieval-dates.md) command to limit the number of events retrieved from the database. This improves performance. Once you've set the retrieval dates, you can further limit the displayed events within the table using the Date picker in this command.
