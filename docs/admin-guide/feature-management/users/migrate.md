# Migrate User

<img width="300" src="/img/admin-guide/feature-management/users/migrate.png"/>

The Migrate User page allows you to migrate a user from one chapter to another.

To migrate a user:
1. Select the user from the User ID dropdown menu.
2. Select the chapter to migrate the user to from the Chapter dropdown menu.
3. Select the user's new postal code from the Postal Code dropdown menu.
<img width="300" src="/img/admin-guide/feature-management/users/migrate-user-1.png"/>
4. Click the `1. Migrate User` button.
<img width="300" src="/img/admin-guide/feature-management/users/migrate-user-2.png"/>
5. Click the `2. Refresh Caches` button to update the chapter caches with the migrated user information.
<img width="300" src="/img/admin-guide/feature-management/users/migrate-user-3.png"/>
6. Click the `3. Set images` button to update the chapter and crop images.

The user will now be migrated to the new chapter with the updated postal code and images.
