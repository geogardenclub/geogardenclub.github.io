# Delete User

<img width="300" src="/img/admin-guide/feature-management/users/delete.png"/>

The Delete User page allows you to delete a user from GGC. Select the user from the dropdown menu. The page will display information about the user.

<img width="300" src="/img/admin-guide/feature-management/users/delete-user-1.png"/>

To delete the user, click the `Delete User` button. This will permanently delete the user and all associated data from GGC.
