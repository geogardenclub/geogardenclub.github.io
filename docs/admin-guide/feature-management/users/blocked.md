# Manage Blocked Users

<img width="300" src="/img/admin-guide/feature-management/users/blocked.png"/>

Blocked users are those who are prevented from accessing GeoGardenClub data
and services because they have violated the [Prohibited Activities clause of our Terms and Conditions](https://geogardenclub.com/docs/user-guide/terms-and-conditions#6-prohibited-activities).

Currently, to manage the set of blocked users, a GGC administrator must manually create, edit, or delete
documents in the blocked_users collection of the GGC Firebase database. 

This page also shows the list of currently blocked users.
