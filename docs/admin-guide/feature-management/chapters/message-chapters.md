# Message Chapters

<img width="300" src="/img/admin-guide/feature-management/chapters/message.png"/>

The Message Chapters page allows you to post Forum messages to the chapters of your choice.

Select the chapters you want to message, choose the forum topic type, the title of the topic, and the message you want to send. You can add an optional image to the topic. Click the **Submit** button to send the message to the selected chapters.
