# Update Chapter

<img width="300" src="/img/admin-guide/feature-management/chapters/update.png"/>

The Update Chapter page allows you to update a chapter. You can change the chapter name, country, state, region, postal codes, and frost dates, if any. You can also choose an image for the chapter. The postal codes field allows you to update the set of postal codes in the chapter. The postal codes field will ensure that there are no overlaps between chapters.
