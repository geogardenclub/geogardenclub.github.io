# Delete Variety

<img width="300" src="/img/admin-guide/feature-management/varieties/delete-1.png"/>

The Delete Variety page allows you to delete an existing variety. If the variety is not associated with any plantings, you can delete it. Click the `Delete` button to delete the variety.

If the variety is associated with any plantings, you will see a warning message and the `Delete` button will be disabled. You will need to update or delete the associated plantings before you can delete the variety.

<img width="300" src="/img/admin-guide/feature-management/varieties/delete.png"/>
