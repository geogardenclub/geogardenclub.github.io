# Update Variety

<img width="300" src="/img/admin-guide/feature-management/varieties/update.png"/>

The Update Variety page allows you to update the name of an existing variety. To update a variety, enter the new variety name in the text box, and click the `Submit` button. The variety name will be updated in the system.
