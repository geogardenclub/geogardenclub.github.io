# Manage Varieties

<img width="300" src="/img/admin-guide/feature-management/varieties/manage.png"/>

The Manage Varieties page uses the familiar GGC management interface. To [create](create) a new variety, click the `+ Varieity` button. To [update](update) or [delete](delete) an existing variety, select the variety from the crop and variety dropdown menus and click the `Update` button.
