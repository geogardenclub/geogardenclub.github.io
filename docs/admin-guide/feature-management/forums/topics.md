# All Forum Topics

<img width="300" src="/img/admin-guide/feature-management/forums/topics.png"/>

The All Forum Topics page lists all the forum topics in GGC. You can see the topic title, the chapter it belongs to, the last update time, and the number of replies. Clicking on the topic tile takes you to the [Forum Messages page](messages) for that topic, where you can view and manage the messages in that topic.
