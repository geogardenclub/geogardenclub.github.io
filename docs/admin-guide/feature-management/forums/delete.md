# Delete Forum Topics

<img width="300" src="/img/admin-guide/feature-management/forums/delete-topics.png"/>

The Delete Forum Topics page allows you to delete forum topics from the system. This includes private messages. Start typing the title of the topic you want to delete. The title search uses lowercase to match the topic titles, so you can use either lower or upper case letters when searching.

<img width="300" src="/img/admin-guide/feature-management/forums/delete-topics-b.png"/>

The page will show you the number of topics and a list of topics that match the title you entered. The titles are abbreviated but you should get an idea of the matches. As you type more characters, the list of matching topics will update.

<img width="300" src="/img/admin-guide/feature-management/forums/delete-topics-bu.png"/>

You can further filter the topics by selecting the created on date.

When you are satisfied with the list of topics, click the **Submit** button. The Confirm Delete Forum Topics page will show you the number of topics that will be deleted and a list of the topics that will be deleted. Click the **Delete** button to delete the topics from the system.

<img width="300" src="/img/admin-guide/feature-management/forums/delete-topics-confirm.png"/>

:::warning Delete Forum Messages
It is very easy to select more topics than you intended to delete. A good practice is to first view all the topics to find the ones you want to delete, remember the title and created on date, and then use that information to filter the topics for deletion.
:::
