# Forum Messages

<img width="300" src="/img/admin-guide/feature-management/forums/messages.png"/>

The Forum Messages page allows you to manage the messages that are displayed in the topic. You can add, edit, or delete messages as needed.

Clicking on the "..." button on the topic message will allow you to edit or delete the topic. Deleting the topic will also delete all messages in that topic.

<img width="300" src="/img/admin-guide/feature-management/forums/topic-popup.png"/>

Clicking on the "..." button on a message will allow you to edit or delete that message.

<img width="300" src="/img/admin-guide/feature-management/forums/message-popup.png"/>
