# Manage Crops

<img width="300" src="/img/admin-guide/feature-management/crops/manage.png"/>

The Manage Crops page is similar to the other Manage pages in Feature Management. It has a dropdown menu to select the crop to [update](crops/update) or [delete](crops/delete), and a `+ Crop` button to [create](crops/create) a new crop.
