# Create Price

<img width="300" src="/img/admin-guide/feature-management/prices/create.png"/>

Select the country, crop, year, and price to create a new price. The price field is an integer field that represents the price in cents. For example, if the price is $1.50, enter `150` in the price field. After filling out the form, click the `Submit` button to create the new price.

**Note:** There can only be one price for a given country, crop, and year. If you try to create a price that already exists, you will receive an error message. In that case, you can either update the existing price or delete it and create a new one.

You can indicate that the crop is priceless by clicking the `Is this crop priceless?` checkbox. If you want to change a priceless crop back to a regular crop, simply uncheck the checkbox and enter a new price.
