# Update Tag

<img width="300" src="/img/admin-guide/feature-management/tags/update.png"/>

The Update Tag page allows you to change the name, description, and parent tag for a tag. After making your changes, click the `Submit` button to update the tag.

The `Parent Tag` dropdown menu allows you to select a parent tag for the tag you are updating. If you want to remove the parent tag, select the `None` option from the dropdown menu.

Clicking the `Delete` icon will take you to the [Delete Tag page](delete).
