# Delete Tag

<img width="300" src="/img/admin-guide/feature-management/tags/delete.png"/>

If the tag you want to delete is not being used in any observations, you can delete the tag by clicking the `Delete` button.

If the tag is being used in observations, you will see a warning message that the tag cannot be deleted. In this case, you can either update the observations to use a different tag or delete the observations that are using the tag before deleting the tag.

<img width="300" src="/img/admin-guide/feature-management/tags/delete-with-observations.png"/>
