# Manage Families

<img width="300" src="/img/admin-guide/feature-management/families/manage.png"/>

The Manage Families page is similar to the other Manage pages in Feature Management. It has a dropdown menu to select the family to [update](families/update) or [delete](families/delete), and a `+ Family` button to [create](families/create) a new Family.
