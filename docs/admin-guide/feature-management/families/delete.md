# Delete Family

<img width="300" src="/img/admin-guide/feature-management/families/delete.png"/>

From the [Update Family page](update), click the **Delete** icon in the title bar to delete the Family.

**Note:** You can't delete Families with crops, so if you have any crops with the Family, you'll need to delete those first.
