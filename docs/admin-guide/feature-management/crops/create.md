# Create Crop

<img width="300" src="/img/admin-guide/feature-management/crops/create.png"/>

You can create a new crop by entering the crop name and selecting the crop's family. Crops are shared across all chapters. If you want to define a chapter local crop name, (e.g. "Ulu" for "Breadfruit"), select the chapter and provide the local name.
