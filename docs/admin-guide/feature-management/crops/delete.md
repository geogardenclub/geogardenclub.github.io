# Delete Crop

<img width="300" src="/img/admin-guide/feature-management/crops/delete-crop-mango.png"/>

From the [Update Crop page](update), click the **Delete** icon in the title bar to delete the crop.

If the crop has no plantings, you can confirm the deletion by clicking the **Delete** button in the confirmation dialog.

**Note:** You can't delete crops with plantings, so if you have any plantings with the crop, you'll need to delete those first. 

<img width="300" src="/img/admin-guide/feature-management/crops/delete.png"/>
