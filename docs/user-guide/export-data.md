# Export Data

The Export Data command provides a way for you to download your Planting data as a CSV file. This is useful, for example, if you wish to leave GeoGardenClub and preserve a copy of your data for future use. 

The Export Data command is only available via the web version of GeoGardenClub. To start, visit https://geogardenclub.app in a browser and login, then tap the Gear icon in the upper right corner of the screen to reveal the "Settings" menu item:

<img width="700" src="/img/user-guide/export-data/settings-dropdown.png"/>

Tapping "Settings" takes you to the Settings screen:

<img width="700" src="/img/user-guide/export-data/settings-screen.png"/>

Next, tap "Export Data" to take you to the Export Data screen:

<img width="700" src="/img/user-guide/export-data/export-data-screen-web.png"/>

Tapping Download Data will download a file to your Downloads directory:

<img width="700" src="/img/user-guide/export-data/export-data-downloads-directory.png"/>

You can open that file in a spreadsheet program of your choice to view the data (not all fields are shown below):

<img width="700" src="/img/user-guide/export-data/export-data-spreadsheet.png"/>

:::warning Data Export is not available on mobile
Note that if you go to the Data Export page on mobile, you will see the following:

<img width="300" src="/img/user-guide/export-data/export-data-screen-mobile.png"/>

:::
