# Settings

:::tip About Settings
:::

The Settings screen allows you to make changes to your look and feel, reset the password, report objectionable content, delete your account, logout, and more.

## Accessing Settings

To access these services, click on the Gear icon (in either mobile or web versions) to display the Settings menu item:

<img width="700" src="/img/user-guide/export-data/settings-dropdown.png"/>


Now select Settings to reveal the Settings screen:

<img width="700" src="/img/user-guide/settings/settings-screen.png"/>
